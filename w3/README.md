## git 명령어 정리
- add: 파일을 추적 대상으로 포함시킬 때, 또는 커밋 대상으로 포함시킬 때 사용
 - 예) git add
- commit: 파일에 대한 정보를 파일에 직접 들어가지 않고 알 수 있게  
 - 예) git commit -m "의미있는 메시지"
- branch: main 말고 다른 저장소
 - 예) git branch branch-name
